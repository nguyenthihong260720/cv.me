## Repository Purpose

This is a document repository for anonymized candidate CVs, maintained under the Keppel Tech / ATS recruitment workflow. It is not a software project — there is no build system, test runner, or application code.

## Contents

- `Annex_A2_Engineer_QA.pdf` — Anonymized CV for Engineer B, a QA Engineer with ~5 years of experience in API and functional testing (Postman, SOAPUI, SQL), currently based in Hanoi, Vietnam.

## CV Format Convention

CVs follow the Keppel Tech / ATS anonymized profile format:
- Header: role designation (e.g., "Engineer B"), specialty, and key stats (experience, primary/secondary stack, English level, location, notice period)
- Sections: Professional Summary, Core Technical Skills, Project Experience, Certifications & Training, Education, Languages
- All personally identifying information is removed; candidates are referred to by letter designation (Engineer A, Engineer B, etc.)
